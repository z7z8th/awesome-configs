## Basic Usage
===
* Mod4 + Enter: Terminal
* Alt + F1:     Terminal
* Alt + F2:     Run command
* Mod4 + Space: Next layout
* Mod4 + [1-5]: Switch to tag [1-5]
* Mod4 + Tab:   Focus next screen
* Alt + Tab:    Focus next client
* Mod4 + j/k:   Focus next/previous client
* Mod4 +m:      Toggle client maximize
